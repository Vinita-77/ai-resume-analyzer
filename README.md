# AI Resume Analyzer

## Overview
This project analyzes resumes using Python and NLP techniques.
It recommends suitable job roles based on skill matching.

## Features
- Upload resume in PDF format
- Skill extraction
- Job role recommendation
- Resume match score
- Missing skill suggestions

## Technologies Used
- Python
- Streamlit
- NLP
- Pandas & NumPy

## How to Run
1. Install dependencies
2. Run: streamlit run app.py
